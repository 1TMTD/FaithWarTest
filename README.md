# FaithWarTest

Тестовий репозиторій для системи **FaithWar**.  
Мета: перевірка пайплайна Survival‑Cloud → GitHub → Colab → Drive → Blender.

## Структура
- `prompts/` — текстові промти для генерації референсів.
- `results/` — збережені картинки.
- `scripts/` — скрипти для Colab та PowerShell.
- `logs/` — логи генерації.

## Використання
1. Локально редагуй `prompts.txt`.
2. Пуш у GitHub.
3. В Colab клонуй репозиторій та запускай генерацію.
4. Результати зберігай у Drive та використовуй у Blender.
